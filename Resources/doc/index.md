Getting started with StewieUserBundle
=====================================

StewieUserBundle provides user management for your Symfony application.

It offers User, Group and Role administration. Roles are persisted into the user object directly via User-Role- and User-Group-Role-assignments. Therefor inheritance in security.yaml is kept to a minimum.
